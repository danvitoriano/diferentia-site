# Claude Code Avançado

**Carga:** ~6h (+ 1h30 opcional de API/Agent SDK)

## Para quem é

Participantes do Vibe Coders que querem Git, MCP no terminal, hooks, slash commands e visão de integração com times de engenharia.

## O que você vai dominar

- Versionamento com Git e fluxo de trabalho com Claude Code
- MCP no terminal (ex.: leitura de dados internos com governança)
- Hooks e automações no ciclo de desenvolvimento
- Módulo opcional: API Anthropic e Claude Agent SDK (workshop conceitual)

## Pré-requisitos

- Claude Introdutório (recomendado)
- Vibe Coders concluído ou equivalente prático
